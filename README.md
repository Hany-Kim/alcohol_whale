# UI 디자인

‘술고래’는 주류에 대해 잘 모르는 입문자를 위한 서비스로 수많은 술이 존재하는 주류 시장을 ‘바다’로, 원하는 술을 찾아주는 안내자를 ‘고래’로 설정하였습니다. 

‘술을 위한 항해’라는 의미를 담아 파란색을 메인컬러로 사용하였다.
